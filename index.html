JAVASCRIPT

javascript
"넷스케이프에서 개발한 스크립트 언어"
프로젝트 적용
  • - 웹 브라우저의 동적인 움직임을
    위해 사용
  • - 서버에서 데이터를 가지고 온 후, 데이터를 제대로 처리해주기 위해 비동기 처리 진행
    비동기란, 자바스크립트에서 특정 코드를 실행한 후,실행이 끝날 때까지 기다리지 않고 다음 내용을 수행합니다. 비동기 처리를 해주지 않으면, 데이터가 불러오지 않았는데도 불구하고 코드를 실행시키기 때문에 문제가 발생할 수 있습니다.

VUE

vue
"애플리케이션을 구축하기 위한 프런트 엔드 JavaScript 프레임워크"
프로젝트 적용
  • - vue는 웹 애플레이션을 개발할 때 도움을 주는 프레임워크로, 해당 프레임워크에서 제공해준 문법을 따라 적용
  • - 컴포넌트 기반으로, 컴포넌트를 분리하여 코드의 재사용성 증가
  • - vuex의 store를 사용하여 데이터를 중앙 관리

NUXT

nuxt
"Vue.js 애플리케이션 개발을 보다 강력하고 사용하기 쉽게 만들어 주는 프레임워크"
프로젝트 적용
  • - SEO(검색 엔진최적화)를 적용시키기 위해 동적으로 meta데이터 처리
  • - SSR(서버 사이드 랜더링)를 위해 사용
    SSR이란, 서버에 html 파일을 저장해두었다가 요청이 발생할 때, 모든 파일을 전달하는 방식입니다.
